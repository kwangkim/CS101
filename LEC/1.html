Declarative vs Imperative

Declarative Knowledge - What

Declarative knowledge is composed of statements of fact.

EX: “y is the square root of x if and only if y*y = x”

Imperative Knowledge - How

Imperative knowledge is about how to accomplish something. Think of it as a recipe.

    1. Start with a guess, g
    1. If g*g is close enough to x => g is a good approximation of the square root of x
    1. Otherwise, create a new guess by averaging g and x/g. I.e., gnew = (g_old + x/g_old)/2
    1. Using this new guess, go back to step 2
